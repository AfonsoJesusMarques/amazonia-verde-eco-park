# Amazônia Verde Eco Adventure Park

This is a Web Technologies assessment project for a promotional eco-adventure wildlife park website.

## Theme

Amazônia Verde Eco Adventure Park is inspired by Brazilian wildlife, rainforest habitats, tropical rivers and conservation experiences.

The website promotes a fictional eco-adventure wildlife park. It does not include ticketing, prices or booking information.

## Technologies

- HTML
- CSS
- JavaScript
- Node.js
- Express
- EJS
- SQLite3

## Current progress

The initial Express application, basic routes, EJS views, public folders and project structure have been created.
